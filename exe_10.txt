Lista de contatos em python

Primeiro o programa daria uma interface para a pessoa entrar com nome de usuario, adicionar, verificar os nomes de usuarios existentes ou sair do programa 
Posteriormente ele iria verificar se existe esse usuario, caso não, retornaria para usuario inexistente e daria a opção de adcionar usuario, entrar usuario ou sair 

caso entre com o usuario correto ele iria trazer uma nova interface para que a pessoa possa adicionar um novo contato, listar os contatos, excluir os contatos ou voltar

caso a pessoa queira adicionar um novo contato, a pessoa deve colocar nome,sobrenome, idade e numero completo do contato. O valor normal para colocar o número de um usúario deve ser de 11 digitos ex [61999999999]
depois após ser validado o programa deve verificar os dois primeiros digitos para saber de qual estado eh e também arrumar na lista de contatos para ficar [(61)9 9999-9999]

temos que garantir que não haja telefones duplicados no mesmo usuario, também temos que garantir para que não haja telefones iguais porém com DDD diferente
